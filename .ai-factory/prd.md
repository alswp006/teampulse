# TeamPulse

## Platform
앱인토스 (Vite + React + TDS)

## One-liner
재택근무 확산으로 팀 결속력이 약해지고 있지만 기존 팀빌딩 솔루션은 시간이 오래 걸리거나 형식적인 문제를 해결하는 비동기 마이크로 팀빌딩 플랫폼

## Problem
한국 IT 기업들이 재택/하이브리드 근무를 도입하면서 팀원 간 유대감이 급격히 떨어지고 있습니다. Zoom 회식이나 MT는 시간이 많이 걸리고 부담스러우며, 형식적인 아이스브레이킹은 실질적 효과가 없습니다. Forbes 조사에 따르면 원격근무 팀의 60%가 '팀 결속력 약화'를 최대 고충으로 꼽았으나, 짧은 시간에 자연스럽게 팀빌딩할 수 있는 도구가 부족합니다.

## Target Users
50인 이상 IT/스타트업 HR 담당자, 팀장급 (30~40대), 재택/하이브리드 근무를 운영하며 팀 문화 유지에 고민하는 조직

## Core Features
1. 매일 5분 마이크로 미션 (팀원끼리 취미 공유, 짧은 칭찬 릴레이, 익명 고민 상담 등)
2. 비동기 참여 가능 (출근 시간 다르거나 시차 있어도 OK)
3. 팀별 참여도 리더보드와 배지 시스템으로 게이미피케이션
4. AI가 팀 분위기 분석해서 적절한 미션 자동 추천 (긴장도 높으면 가벼운 미션)
5. 주간 팀 리포트 (참여율, 긍정 키워드 분석, 팀 분위기 점수)

## Monetization
- Model: B2B SaaS subscription
- Estimated MRR: $8,000
- Reasoning: 팀당(10~50명) 월 99,000원 요금제. 한국 IT 기업 약 5만개 중 재택근무 도입 기업 50%, 그 중 0.5%만 도입해도 125개 팀. 평균 팀당 월 10만원으로 월 1,250만원(약 $10,000). HR 예산에서 팀빌딩 비용은 일반적으로 1인당 연 10~20만원 책정되므로, 월 1만원/인 수준은 매우 합리적. 무료 체험 2주 제공 후 전환율 30% 가정하면 월 $8,000 가능.
